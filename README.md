# Erarta-Artwork-Scraper

Этот проект содержит два инструмента для автоматизированного сбора информации о произведениях искусства с сайта Музея современного искусства «Эрарта». Скрипты демонстрируют работу с динамическим контентом с помощью Python и Selenium.

## Описание

Проект включает два этапа:

1. Сбор ссылок на все карточки произведений.
2. Обход этих ссылок и извлечение информации с каждой страницы.

Код предназначен только для учебных целей и не содержит выгруженных данных.

## Структура проекта

### link_scraper.py
Собирает ссылки на все произведения, проходя по страницам пагинации каталога.  
Результат — файл `art_links_all.csv` со столбцом `url`.

### link_visitor.py
Загружает каждую страницу по списку ссылок и извлекает:
- название;
- автора;
- цену;
- размер;
- год;
- тип;
- жанр;
- багет (если есть);
- технику.

Результат — файл `art_org.csv`.
